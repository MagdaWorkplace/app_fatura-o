# app_fatura-o
# App de leitura de faturas com o QR Code

# Descrição:
# App para ler faturas com QR code , exportar os dados e enviar para uma lista de movimentos de saída, para posteriormente enviar para uma api de movimentos de saída junto com a foto da fatura.

# Tecnologias a utilizadar:
# -> Python
# -> Backend: FastAPI
# -> Frontend: HTML + CSS + JS
# -> Base de dados: SQlite
# -> Leitura de QR code: Pyzbar + OpenCV
# -> Autenticação: JW (Json Web Token)
# -> Addicionais: 

# Fluxo da aplicação - Versão inicial
#
# 0. Registar
# 1. Login do Utilizador
# 2. Upload da fatura(imagem)
# 3. Leitura do QR code
# 4. Extração e organização dos dados
# 5. Criação do movimento de saída
# 6. Armazenamento na base de dados 
# 7. Envio para API - Simulação
#
